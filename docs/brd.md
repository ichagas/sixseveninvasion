# BDR – 6-7 Invasion (Simple Version)

## 1. Purpose
Create a small, humorous tap-based mobile/web game where players spread the “6-7” meme to grow their influence and unlock new locations.

## 2. Goals
- Provide quick, addictive gameplay.
- Deliver a simple upgrade system.
- Include light animations and sound effects.
- Support iOS, Android, and Web.

## 3. Key Features
- Tap to generate “6-7 Energy.”
- Passive generators (bots, kids, echo chambers).
- Location progression (Classroom → Gym → Street → Internet).
- Resistance events (teachers, parents, algorithm).
- Simple prestige system (“Clout”).

## 4. Audience
Casual players, meme lovers, short-session gamers.

## 5. Success Metrics
- 1–3 minute fun loops.
- High replayability via upgrades.
- Smooth performance across platforms.
